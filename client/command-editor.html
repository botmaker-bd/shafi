Welcome Message

Display available commands

Repeat user's message

Answer Handler Templates

Echo Answer

Simply repeat user's answer

const answer = getAnswer(); sendMessage(`You answered: ${answer}`);

Validate Answer

Check if answer meets criteria

const answer = getAnswer(); if (answer.length < 3) { sendMessage("Answer too short. Please try again."); } else { sendMessage("Thank you for your answer!"); }

Process Data

Process and store user input

const answer = getAnswer(); // Process the answer here sendMessage(`Data received: ${answer}`); // Store or process the data as needed
